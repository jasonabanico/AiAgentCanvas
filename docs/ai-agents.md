# AI Agents

AI Agent Canvas is a multi-agent enterprise copilot platform built on Microsoft Agent Framework, Azure AI Foundry, and CopilotKit. This guide covers the core concepts, protocols, and patterns that power the platform.

| [Tools and Skills](ai-agents-tools.md) | AITool/AIFunction, DynamicToolRegistry, tool providers, and agent-managed skills |
| [Model Context Protocol](ai-agents-mcp.md) | MCP overview, dynamic connections, tool discovery, and security considerations |
| [Context Providers](ai-agents-context.md) | The context injection chain, built-in providers (persona, RAG, guardrails, governance), and custom providers |
| [AG-UI Protocol](ai-agents-protocol.md) | SSE streaming, the /api/copilotkit endpoint, event types, and CopilotKit integration |
